OpenJS
=============
OpenJS是腾讯微博开放平台提供的Javascript库，开发者使用该库可以轻松完成腾讯微博用户授权，访问API接口，调用官方组件等功能，支持多种应用部署环境及无缝迁移。

贡献代码
--------
1. 创建一个fork
2. clone到本地

        git clone https://github.com/michalliu/open-js.git
3. 更新submodule
       
        git submodule init
        git submodule update

4. 在github上发起pullrequest
